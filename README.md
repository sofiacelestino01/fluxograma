# Flowgorithm
**Meu repositório de fluxograma**

No primeiro exercício, o fluxograma realiza uma interação inicial com o usuário. Ele lê o nome de uma pessoa e diz Olá

No segundo exercício, o fluxograma trabalha com soma. Ele pede dois números ao usuário, realiza a soma entre eles e apresenta o resultado.

No terceiro exercício, o fluxograma lê a idade de uma pessoa e verifica se ela é maior ou menor de idade. Caso a idade seja maior ou igual a 18, o sistema fala que a pessoa é maior de idade; caso contrário, fala que é menor.

No quarto exercício, o algoritmo realiza uma classificação numérica. Ele lê um número e determina se ele é positivo, negativo ou igual a zero.

No quinto exercício, o fluxograma volta o uso de operações matemáticas, agora com o cálculo de média. Ele lê duas notas, calcula a média entre elas e exibe o resultado final.
